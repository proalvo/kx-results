1a. Split-time TT Timing (optional)

For Time Trials timed with stopwatches on one shared running clock (no per-athlete chronometer): athletes start at a fixed interval, offset by a time shift so athlete 1 doesn't start at 0:00. At the finish line, the raw stopwatch reading (the "split") is entered instead of a direct time, and the actual result time is calculated automatically. Leave both fields blank to disable — Time is then entered directly as usual, the default for every competition. Applies to the active competition (above).

4. Publish to website (optional)

Results can be published to the public results website so spectators can follow online. The website address and the organization key are configured once per server — the same website is used for all competitions. Each competition is then connected individually; the website issues it a competition-specific key automatically. Once connected, results sync in the background whenever they change.
