# Event with 6 athletes

File: rule_SMSL2023_6-athletes.json

- 2 heats in Semi-Final
- Two from each heat advance to the Final.


```

TIME TRIAL (TT)
  Rank 1–6
      │────────────────────────────────│
      ▼                                ▼
SEMI-FINAL 1 (SF1)              SEMI-FINAL 2 (SF2)
─────────────────              ─────────────────
TT 1  → Slot 1                 TT 2  → Slot 1
TT 4  → Slot 2                 TT 3  → Slot 2
TT 5  → RESULT 5-6             TT 6  → RESULT 5-6

      │────────────────────────────────│
      ▼
FINAL
─────────────────
Slot 1 ← SF1 1st
Slot 1 ← SF2 1st
Slot 3 ← SF1 2nd
Slot 4 ← SF2 2nd
      │
      ▼
RESULT
─────────────────
1 ← F 1st  →  1
2 ← F 2nd  →  2
3 ← F 3rd  →  3
4 ← F 4th  →  4
5 ← SF1-2 3rd, faster in TT
6 ← SF1-2 3rd, slower in TT

```
