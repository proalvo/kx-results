# Rules

This archive has pre-defined rules, but you can create also your own set of rules. 

Rules can by uploaded to the database with command:
```
node scripts/upload-rules.js
```
Please, name the files as follows:
`rule_<association><year_or_rules>_<number_athletes_in_this_rule_file>-athletes.json`

SMSL - stands for Suomen melonta-ja soutuliitto (Finnish canoing and rowing association).
